<interfaces>
<!-- API contracts from Wave 2 routes -->

GET /api/admin/uploads → { uploads: Upload[] }
Upload.status: 'processing' | 'indexed' | 'error'
Upload.error_message: string | null (shown in error badge)
Upload.file_type: 'csv' | 'pdf'
Upload.row_count: number | null (for CSV)
Upload.chunk_count: number | null (for PDF)

POST /api/admin/upload → FormData with 'file' field
  CSV → 201 { uploadId, rowCount }
  PDF → 202 { uploadId }
  Error → 400 { errors: string[] } or { error: string }

DELETE /api/admin/uploads/[id] → { success: true } or 404

GET /api/admin/stats → { stats: ListingStatsData | null }
ListingStatsData.total_listings: number
ListingStatsData.count_by_status: { available: number, sold: number, reserved: number }

Template CSV columns (for download link):
property_id, address, area, price_aed, bedrooms, bathrooms, size_sqft, status, developer, community, property_type, sold_date

Polling pattern (from RESEARCH.md):
```typescript
useEffect(() => {
  const hasProcessing = uploads.some(u => u.status === 'processing')
  if (!hasProcessing) return
  const id = setInterval(fetchUploads, 5000)
  return () => clearInterval(id)
}, [uploads])
```
