    - requestWakeLock() when navigator.wakeLock is absent: sets needsManualWarning = true
    - requestWakeLock() when navigator.wakeLock.request() throws: sets needsManualWarning = true
    - requestWakeLock() when request() succeeds: sets wakeLockActive = true, needsManualWarning = false
    - WakeLock 'release' event fires: sets wakeLockActive = false, needsManualWarning = true (screen locked warning)
    - releaseWakeLock() clears both flags

    useVisibilityGuard:
    - When document.visibilityState changes to 'hidden': calls onHide()
    - When document.visibilityState changes to 'visible': calls onShow() (NOT auto-resume)
    - Cleans up event listener on unmount

    POST /api/session/start:
    - Creates session record with tenantId from getServerSupabase(), consent_confirmed_at from request body
    - Returns 201 with { sessionId }

    POST /api/session/end:
    - Accepts { sessionId, durationSeconds }; validates with Zod
    - Updates session status to 'ended', sets ended_at = now(), duration_seconds
    - If durationSeconds < MIN_SESSION_DURATION_SECONDS: returns 200 with { ok: true, shortSession: true }

    POST /api/session/pause:
    - Accepts { sessionId, action: 'pause' | 'resume' }; validates with Zod
    - Updates session status accordingly
  </behavior>
